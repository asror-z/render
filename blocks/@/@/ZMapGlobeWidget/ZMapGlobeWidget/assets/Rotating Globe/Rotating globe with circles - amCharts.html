Rotating globe with circles

With amCharts 4 you can create amazing Map charts and you don’t have to be limited to flattened representation of Earth. As you can see in this demo, you can use an orthographic projection to display the planet as an interactive globe.

Learn more about map projections in our documentation.

    }

    }

  })
