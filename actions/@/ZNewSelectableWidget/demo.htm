Selectable

Quickly select HTML DOM elements or groups of elements.

Introduction

Selectable allows HTML DOM elements to be selectable by way of clicking / tapping on the elements or clicking and dragging to draw a selection box.

Features

  • No dependencies.
  • Lightweight - less than 5kB gzipped!
  • Compatible with all modern browsers - including IE9.
  • Touch enabled for use with mobile devices such as tablets and smartphones.
  • Customizable.
  • Fluent API.

Demos

Click and / or drag to select items. Hold Ctrl to select multiple. Hold Shift to select consecutive groups of items.
