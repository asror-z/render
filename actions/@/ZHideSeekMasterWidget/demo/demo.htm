jQuery HideSeek Plugin Examples

A simple, yet customizable live search jQuery plugin


downloador view on GitHub

HideSeek is simple, yet customizable jQuery plugin for for live search, is compatible with IE7+, Chrome, Firefox, Safari and jQuery 1.8.1, 1.9.1, 1.10.1, 2.0.2. and supports highlighting, custom message and initialization via data attribute.
