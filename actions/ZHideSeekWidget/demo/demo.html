HideSeek

A simple, yet customizable live search jQuery plugin

download or view on GitHub

HideSeek is simple, yet customizable jQuery plugin for live search, is compatible with IE7+, Chrome, Firefox, Safari and jQuery 1.8.x, 1.9.x, 1.11.x, 1.12.x, 2.1.x, 2.2.x. and supports highlighting, custom message, navigation, initialization via data attribute, custom events and more.
