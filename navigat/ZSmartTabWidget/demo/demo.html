jQuery Smart Tab

The flexible jQuery tab control plugin

jQuery Smart Tab is a jQuery plugin for easy implemetaion of tab interface. Flexible and easy to implement with a lot of features that you will love like Ajax contents, verity of themes, cool animation effects, event support, easy to customize.

Features

  • Bootstrap compatible

  • Cool transition animations (fade/slide-horizontal/slide-vertical/slide-swing)
  • External anchor support

CSS Files Selection

You can see many CSS files at dist/css/ folder and may be confused about which one to use. The answer is you can use any of them, all the CSS files are standalone.

The only difference in the files is the theme. The file ../dist/css/smart_tab_all.css contains all the themes, while the ../dist/css/smart_tab.css only contains the default theme. Also other files are theme-specific file named with the respective theme names, like ../dist/css/smart_tab_(theme name).css.

All these files have corresponding min files that compressed for size reduction, those files are recommended unless you don't want to customize the CSS yourself. Ex. ../dist/css/smart_tab_all.min.css

Advanced Options

with all optional parameters
  1. $('#smarttab').smartTab({
  2.   selected: 0, // Initial selected tab, 0 = first tab
  3.   theme: 'default', // theme for the tab, related css need to include for other than default theme
  4.   orientation: 'horizontal', // Nav menu orientation. horizontal/vertical
  5.   justified: true, // Nav menu justification. true/false
  6.   autoAdjustHeight: true, // Automatically adjust content height
  7.   backButtonSupport: true, // Enable the back button support
  8.   enableURLhash: true, // Enable selection of the tab based on url hash
  9.   transition: {
  10.       animation: 'none', // Effect on navigation, none/fade/slide-horizontal/slide-vertical/slide-swing
  11.       speed: '400', // Transion animation speed
  12.       easing:'' // Transition animation easing. Not supported without a jQuery easing plugin
  13.   },
  14.   autoProgress: { // Auto navigate tabs on interval
  15.       enabled: false, // Enable/Disable Auto navigation
  16.       interval: 3500, // Auto navigate Interval (used only if "autoProgress" is set to true)
  17.       stopOnFocus: true, // Stop auto navigation on focus and resume on outfocus
  18.   },
  19.   keyboardSettings: {
  20.       keyNavigation: true, // Enable/Disable keyboard navigation(left and right keys are used if enabled)
  21.       keyLeft: [37], // Left key code
  22.       keyRight: [39] // Right key code
  23.   }
  24. });
Implement Ajax Contents

Ajax is not built-in, instead you can listen to tabContent event and provide the ajax contents. This is usefull to load ajax contents and give you full control on the ajax calls. The tabContent event can accept Promise as return or you can return the string as the content of the tab. The event will be passed with two parameters, anchorObject the jQuery object of the selected tab and tabIndex the zero based index number of the tabs. Here is an ajax call example with Promise. 

  1. // Ajax content loading with "tabContent" event
  2. $("#smarttab").on("tabContent", function(e, anchorObject, tabIndex) {
  3.  
  4.     var repo    = anchorObject.data('repo');
  5.     var ajaxURL = "https://api.npms.io/v2/package/" + repo;
  6.  
  7.     // Return a promise object
  8.     return new Promise((resolve, reject) => {
  9.  
  10.       // Ajax call to fetch your content
  11.       $.ajax({
  12.           method  : "GET",
  13.           url     : ajaxURL,
  14.           beforeSend: function( xhr ) {
  15.               // Show the loader
  16.               $('#smarttab').smartTab("loader", "show");
  17.           }
  18.       }).done(function( res ) {
  19.  
  20.           var html = 'Ajax data about ' + repo + ' loaded from GitHub';
  21.           html += 'URL:' + ajaxURL;
  22.           html += 'Name: ' + res.collected.metadata.name;
  23.  
  24.           // Resolve the Promise with the tab content
  25.           resolve(html);
  26.  
  27.           // Hide the loader
  28.           $('#smarttab').smartTab("loader", "hide");
  29.       }).fail(function(err) {
  30.  
  31.         // Reject the Promise with error message to show as tab content
  32.         reject( "An error loading the resource" );
  33.  
  34.         // Hide the loader
  35.         $('#smarttab').smartTab("loader", "hide");
  36.       });
  37.  
  38.     });
  39.  
  40. });
  41.  
  42.  
  43. // SmartTab initialize
  44. $('#smarttab').smartTab();
Public functions

There are public methods that are handy to do certain functions

  13.

  19. };

loader

Allow you to show/hide the built-n loader animation. 

  1. // Show the loader
  2. $('#smarttab').smartTab("loader", "show");
  3.  
  4. // Hide the loader
  5. $('#smarttab').smartTab("loader", "hide");

Parameter Description Default
selected (integer) Specify the selected tab on the first load 0

orientation (string) Nav menu orientation 'horizontal'

autoAdjustHeight (boolean) Automatically adjust content height true
backButtonSupport (boolean) Enable the back button support true
enableURLhash (boolean) Enable selection of the tab based on url hash true
transition (object) with settings for the transition

animation (string) Effect on navigation, none/fade/slide-horizontal/slide-vertical/slide-swing 'none'
speed (number) Transion animation speed '400'
easing (string) Transition animation easing. Not supported without a jQuery easing plugin ''
autoProgress (object) with settings for the auto navigate tabs on interval

enabled (boolean) Enable/Disable Auto navigation false
interval (number) Auto navigate Interval (used only if "autoProgress" is enabled) 3500
stopOnFocus (boolean) Stop auto navigation on focus and resume on outfocus (used only if "autoProgress" is enabled) true
keyboardSettings (object) with settings for the keyboard navigation

keyNavigation (boolean) Enable/Disable keyboard navigation true
keyLeft (array) Left key code [37]
keyRight (array) Right key code [39]

Event Description

Event Description Parameters
leaveTab Triggers when leaving a tab.
This is a decision making event. based on its function return value (true/false) the current navigation can be cancelled. 		Object: jQuery object of the tab anchor element.
Int: Index of the tab.
showTab Triggers after a tab is loaded with content. Object: jQuery object of the tab anchor element.
Int: Index of the tab.
tabContent Triggers when a tab is about to load.
This event will allow you to pass a string value as return parameter. The string value will be used as the content of the tab. You can also pass a Promise object and on Promise completion the result will be used as the content of the tab. This is usefull to load ajax contents and give you more control on the ajax calls. 		Object: jQuery object of the tab anchor element.
Int: Index of the tab.

Event Initialize Examples

  1. // Initialize the leaveTab event
  2. $("#smarttab").on("leaveTab", function(e, anchorObject, tabIndex) {
  3.    return confirm("Do you want to leave the tab "+tabIndex+"?");
  4. });
  5.  
  6. // Initialize the showTab event
  7. $("#smarttab").on("showTab", function(e, anchorObject, tabIndex) {
  8.    alert("You are on tab "+tabIndex+" now");
  9. });
  10.  
  11. // Initialize Smart Tab
  12. $('#smarttab').smartTab();
